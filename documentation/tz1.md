Нужна программа для рисования схем на сетке в клеточку, где рисунок (его отдельные линии)идет по линиям между клетками, горизонтальными и вертикальными.
То есть, юзер не закрашивает клетку целиком, а только границу клетки клеток.

Дизайн
- Современный легкий дизайн в одну страницу
- Сочетание клетки и границы должно хорошо читаться и быть приятным глазу

Рисование 
- происходит на канве, 
- канва расчерчена клетками тонкими серыми линиями, 
- при первом открытии сервиса размер канвы 20х20. 
- имеется разметка - номера клеток (по горизонтали и по вертикали)
- чуть утолщенными но бледными линиями линиями разметка по 10 клеток (как на миллиметровке)
- по клику левой кнопки мыши на границе между клетками ставится линия. Должны быть определенная толерантность в 1/4 клетки, чтобы мышкой не надо было попадать точно
- по правой кнопки мышки линия стирается
- при левом клике по канве с shift протягивается горизонтальная либо вертикальная линия от кликнутой линии до ближайшей закрашенной границе. Если в направлении протягивания нет закрашенной границы - ничего не происходит. Горизонталь/вертикаль определяется по той области, где кликнул юзер - по вертикали или горизонтали. Shift+правый клик пока не делает ничего.
- при клике по канве с ctrl все в пределах очерченных границ заполняется линиями, горизонтальными и вертикальными. Границами считается замкнутый контур из линий. Если контур не замкнут - появляется всплывающее сообщение на пару секунд "заливка невозможна, контур не замкнут". Заливка происходит от точки клика (flood fill) стандартно, по линиям между клетками. Ctrl+правая кнопка - пока ничего не происходит.
- канва масштабируется по колесу мышки. Максимальный предел масштаба - 1 клетка уменьшается до 8 пикселей на клетку. Минимальный - 1 клетка увеличивается до 50 пикселей. Масштабирование идет относительно курсора
- при вождении мышкой с зажатой левой клавишей происходит рисование линий
- при вождении мышкой с зажатой правой клавишей происходит стирание линий
- можно раскрасить цветом клетки (не линии между ними, а именно клетки) в отдельно выделенных зонах только горизонтально на всю ширину или только вертикально на всю высоту поля. Для этого нужно кликнуть на номер столбца/строки. Клик на номер строки → закрашивается вся строка из клеток (не из линий!) целиком выбранным цветом. Про цвета указано ниже в разделе меню. По правому клику на номер строки закраска линии снимается и ряд (столбец) приобретает дефолтный цвет. Разные строки/столбцы можно закрасить разными цветами, при этом перекрашиваемая клетка приобретает новый цвет. Например, если закрасить строку красным, а столбец синим, то на пересечении будет последний применённый цвет.
- можно нарисовать или стереть линию между клетками но всю высоту или ширину канвы, если кликнуть левой или правой клавишей мышки на "хвост" линии там, где идет нумерация строк и столбцов. Хвосты линий - это продолжения линий сетки за границами сетки, там, где идет нумерация. Нумерация идет с 0.

Слева убирающееся меню, в котором выбираются функции и есть информеры
- Undo / Redo
- информер о статистике:
  - размер канвы
  - максимальная высота и ширина рисунка - по сути bounding box нарисованных линий (от минимальной до максимальной координаты)
  - количество выставленных горизонтальных и вертикальных линий
- раскрывающийся информер с горячими клавишами
 - Shift+click - протянуть линию
 - Ctrl+click - залить все линии в пределах границ
- резет (очищение поля, размер остается)
- задать размеры канвы (варианты - имеющийся рисунок сохраняется по центру, либо по желанию пользователя канва добавляется с любого края, аналогично при уменьшении размера канвы рисунок обрезается). Интерфейс этого раздела - "крест" из окошек с инпутами (предзаполнеными числами 10). Около каждого окошка кнопки + и - - для добавления и удаления клеток. В середине кнопки + и - для добавления со всех сторон. Общий дизайн должен быть легким и понятным.
- Выбрать цвет (предложим юзеру выбрать из 8 приятных, но контрастных цветов, можно использовать палитру Google). При этом для закраски клеток используется выбранный цвет, но более бледный, так, чтобы граница клетки, разметка и сама клетка хорошо визуально отличались бы. Можно рисовать линии разными цветами. Выбрал цвет в меню → все новые линии рисуются этим цветом. Можно перекрасить уже нарисованную линию (кликнуть по ней другим цветом). 
- функция печати. При этом программа предлагает в зависимости от размера печать в 1 pdf файл, либо в 2-4 и более файлов, которые потом можно будет распечатать на листе и объединить в схему. При печати в несколько страниц страницы должны быть пронумерованы. Печать на лист А4, одна клетка - 7 мм.
- выгрузка в json файл. Структура на усмотрение разработчика, наиболее подходящая для такой задачи.
- загрузка из json файла
